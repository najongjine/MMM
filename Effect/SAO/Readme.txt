//////////////////////////////////////////////////////////////////////////////
// Ambient Obscurance for MMMv2
// 2015/08/24 Mogg
//////////////////////////////////////////////////////////////////////////////

MikuMikuMoving version2用のAmbient Obscuranceエフェクトです。

■使い方
==============================================================================
ScalableAO.effをMMMv2で読み込んで下さい。

■パラメータ
==============================================================================
Scale
スクリーン空間の大きさを表します。

Radius
アンビエント項を判定する半径です。
大きければ大きいほど、周りの構造に影響を受けます。

Intensity
明度です。アンビエントの濃さを表します。

Bias
判定にかけるバイアス値です。


■参照情報
==============================================================================
"Scalable Ambient Obscurance"
Morgan McGuire (NVIDIA), Michael Mara (NVIDIA), David Luebke (NVIDIA), in High Performance Graphics, June 2012 
https://research.nvidia.com/publication/scalable-ambient-obscurance
